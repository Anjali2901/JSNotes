Introduction

JavaScript is a light-weight, cross-platform, object-oriented language. Unlike C++ and Java, JavaScript is not a compiled language. It is an interpretted language like python, which means the JS code is executed line by line and if any error shows up, the interpretter stops there. JS is used for providing the control of HTML and CSS to the browser.

JS code can be interpretted on google-chrome or mozilla-firefox. For this tutorial, Google-chrome or Chromium-browser is preffered. Just press ctrl+shit+i and then go to console tab. You can either execute JS code line by line here or you can write the JS code in your favourite editor and then paste it here in the console. Now its time for the hello-world ritual. To print anything in the console, the syntax is:

console.log("Hello, World!!");

Also checkout the syntax below. It is used to create alerts.

alert("Message that you want to display");

By now you might have noticed undefined that gets printed on the console after every command. The console always prints the whatever the last command returns. The print and alert functions don't return anything, so, undefined is printed.

Including javascript in a web page

The tag script is used for including JS within a web page.

<script src="/path_to_file/file.js" type="text/javascript"></script>

For including the JS inside the HTML document, the JS code must be enclosed within the script tags and the src attribute should not be mentioned.

Operators

Arithmetic Operators

Javascript supports basic arithmetic operations like Addition (+),subtraction (-),multiplication (*), division (/) and modulus (%).

console.log(13+12); console.log(13-12); console.log(5*3); console.log(16/4); console.log(15%4);

on console..

-> 25 -> 1 -> 15 -> 4 -> 3

Assignment Operators

The Assignment operator = assigns a particular value to a variable.

var x=3; //3 is assigned to x

Addition assignment:

var x = 10; x +=5; //x=x+5; console.log(x);

on console..

-> 15

Multiplication assignment:

var y=10; y*=5; //y=y*5; console.log(y);

on console..

-> 50

In general..

x = y (x = y) //Assignment x += y (x = x + y) //Addition assignment x -= y (x = x - y) //Subtraction assignment x *= y (x = x * y) //Multiplication assignment x /= y (x = x / y) //Division assignment x %= y (x = x % y) //Remainder assignment x **= y (x = x ** y) //Exponential assignment x <<= y (x = x << y) //Left shift assignment x >>= y (x = x >> y) //Right shift assignment x >>>= y (x = x >>> y) //Unsigned right shift assignment x &= y (x = x & y) //Bitwise AND assignment x ^= y (x = x ^ y) //Bitwise XOR assignment x |= y (x = x | y) //Bitwise OR assignment

Comparison Operators

The Comparison operator used to compare two values.

if ('5'==5) return true; else return false;

on console..

-> true

But if we try === operator instead of == it will return false as the === operator strictly compares the two values.

console.log(3>4); //-> false console.log(3!=4); //-> true console.log('5'!=5); //-> false console.log('5'!==5); //-> true

in general..

== equal to === strictly equal to != not equal !== strictly not equal > greater than < less than >= greater than or equal to <= less than or equal to

Bitwise Operators

A bitwise operation operates on one or more bit patterns or binary numerals at the level of their individual bits.

console.log(1&0); //-> 0 console.log(1&1); //-> 1 console.log(!true); //-> false

in general..

& AND ~ NOT ^ XOR << shift left >> shift right >>> Zero-fill right shift

some more operations..

console.log(7^3); //-> 4 console.log(9&1); //-> 1 console.log(5|2); //-> 7

Note:In case of bitwise operators the operands first converted to binary and then bitwise operations are performed b/w the converted binary numbers.

console.log(7^3); 7 -> 0 1 1 1 3 -> ^ 0 0 1 1 ------------------- result-> 0 1 0 0 //-> 4 console.log(9&1); 9 -> 1 0 0 1 1 -> & 0 0 0 1 ------------------- result-> 0 0 0 1 //-> 1 console.log(5|2); 5 -> 0 1 0 1 2 -> 0 0 1 0 ------------------- result-> 0 1 1 1 //-> 7

String Operator

In javascript the + operator is used to concatenate two strings.

var x = "The Big"; var y = "Bang Theory!"; console.log(x+" "+y); // -> The Big Bang Theory! var actor = "Robert" actor+=" Downey Jr." console.log(actor); // -> Robert Downey Jr.

Logical Operators

Logical operators return true or false depending upon the value of operands.

There are three kinds of logical operators.

  • && (logical AND)
  • || (logical OR)
  • ! (logical NOT)
console.log(true&&true); // -> true console.log(false&&true&&true); // -> false

Note:While using Logical AND if any of the operand value is false then the resultant expression will return false else it will return true.

console.log(true||true); // -> true console.log(true||true||false); // -> true console.log(false||false); // -> false

Note: In case of Logical OR if any of the operand value is true then the expression also returns true else it will return false.

console.log(!true); // -> false console.log(!false); // -> true

Conditional Operator

console.log(4<3?first:second); // -> second console.log(4>3?first:second); // -> first

In general..

condition ? first : second

Conditional operator also known as ternary operator returns first if the condition is true else returns second if the condition is false.

For, While

For performing an action again and again loops are used in javascript.

The "for" loop.

for(initialisation;terminationCondition;increment/decrement) { // action }

The first value is the initialisation condition(from where loop starts),the second one is the termination condition(where loop ends) and the third one is the increment/decrement condition(the value incremented or decremented by the provided value).

for(var i=0;i<5;i++) { console.log("for loop prints: "+i); }

on console..

"for loop prints: 0" "for loop prints: 1" "for loop prints: 2" "for loop prints: 3" "for loop prints: 4"

nested "for" loops..

for (var i = 1; i <= 5; i++) { var output = ""; for (var j = 1; j <= i; j++) { output += "*" } console.log(output); }

on console..

"*" "**" "***" "****" "*****"

The "for-in" loop

The for-in loop iterates through the properties of an object.

var iceCream = {name:"Vadilal", flavour:"ButterScotch", type:"Cornetto"}; var taste = ""; var x; for (x in iceCream) { taste += iceCream[x]+ " "; } console.log(taste);

on console..

"Vadilal ButterScotch Cornetto "

Vadilal ButterScotch Cornetto are the properties of the iceCream object.

Javascript objects will be discussed in the later section.

The "While" loop

The while loop works until a condition returns false.

var i=5; //initialisation while(i>=1) //termination condition { console.log("While loop prints: "+i); i--; //decrement condition }

on console..

"While loop prints: 5" "While loop prints: 4" "While loop prints: 3" "While loop prints: 2" "While loop prints: 1"

The "Do-while" loop

var test=" "; var i=0; do { test += "Do-While prints " + i+'\n'; i++; } while (i < 5); console.log(test);

on console..

" Do-While prints 0 Do-While prints 1 Do-While prints 2 Do-While prints 3 Do-While prints 4 "

in general..

do statement while (condition);
